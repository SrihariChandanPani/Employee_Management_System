# Employee_Management_System
This is a full-stack Employee Management System built with Spring Boot (backend), ReactJS (frontend), and MySQL (database). It demonstrates a complete CRUD (Create, Read, Update, Delete) functionality for managing employee records.
Tech Stack::
Frontend: ReactJS, Axios, Bootstrap
Backend: Spring Boot, Spring Data JPA, REST API
Database: MySQL
Features::
Add new employees
View all employees
Update employee details
Delete employee records
